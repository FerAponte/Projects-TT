📊 A/B Testing: Optimización de Landing Page
🧠 Objetivo

Evaluar qué versión de landing page (A vs B) maximiza la conversión y el valor por usuario.

🔍 Análisis realizado

Validación de datos
Test de conversión (Z-test)
Test de gasto (t-test)
Análisis por canal de tráfico
Segmentación por tipo de usuario

📈 Resultados clave

La versión B incrementa la conversión (15.96% vs 12.57%)
Diferencia significativa en gasto promedio
Email y Ads convierten mejor
No hay diferencia entre usuarios nuevos y recurrentes


💡 Recomendaciones

Implementar versión B
Priorizar canales de alto rendimiento
Estrategia unificada de usuarios
